# Security Policy

## Reporting a vulnerability

Please report suspected security vulnerabilities, authentication concerns, or misuse of Mosaic CrossStream privately to [cl271@outlook.com](mailto:cl271@outlook.com).

Include a clear description, reproduction steps, affected version, and any relevant screenshots or logs. Do not include passwords, access tokens, refresh tokens, or other account secrets.

## Supported releases

The project is actively developed. Security fixes are applied to the current supported release line when a verified issue is found.
